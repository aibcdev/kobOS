# Owner.com crawl output

Generated by:

```bash
# Free — no Browserbase (recommended for Owner marketing pages)
npm run crawl:owner:free

# Paid — full browser render (only if fetch misses content)
npm run crawl:owner
```

Each run creates a timestamped folder with:

- `pages/*.html` — rendered HTML from Browserbase
- `pages/*.json` — title, meta, H1s, link counts
- `manifest.json` — all pages
- `SUMMARY.md` — quick index

**Options**

- `npm run crawl:owner -- --dry-run` — list URLs only
- `npm run crawl:owner -- --max=10` — limit pages (default 30)
- `OWNER_CRAWL_MAX=50 npm run crawl:owner` — env override

Requires `BROWSERBASE_API_KEY` and `BROWSERBASE_PROJECT_ID` in `.env.local`.
